05

The Moat

Three capabilities
nobody else has.

🔁

Loop Detection

Pattern-based: same tool + same args repeated N times. Cycle detection (A→B→C→A). Output stagnation.

Closest: agent-loop-guard (0 stars, library only)

💰

Budget Enforcement

Per-session and per-tool cost limits. Auto-kills sessions on breach. Not just tracking — enforcement.

Closest: Preloop (9 stars, very early)

⚡

Circuit Breakers

Tool-level. Closed → Open → Half-open. Auto-recovery with probe. Stateful across invocations.

Closest: AgentArmor (0 stars, library only)

Every framework has a dumb max_iterations counter. Nobody has intelligent pattern detection.
