## What you'll build

By the end of this page you'll have LangSight running locally and capturing
every tool call your AI agent makes — with full traces visible in both the
CLI and the dashboard.

## Prerequisites

- Python 3.11+
- Docker Desktop (recommended) — or skip Docker for CLI-only use

---

## 2. Instrument your agent

Add 3 lines to your existing agent code. LangSight wraps your MCP client
and captures every `call_tool()` automatically — fire-and-forget, fail-open.

```python
from langsight.sdk import LangSightClient

# Connect to your LangSight server
langsight = LangSightClient(url="http://localhost:8000")

# Wrap your MCP session — every call_tool() is now traced
traced = langsight.wrap(
    mcp_session,                     # your existing MCP client
    server_name="postgres-mcp",      # which server this is
    agent_name="my-agent",           # which agent is calling
    session_id="sess-001",           # groups calls into one session
)

# Use traced exactly like your original session — the API is identical
result = await traced.call_tool("query", {"sql": "SELECT 1"})
#        ↑ this call is traced automatically
```

<Note>
  `wrap()` returns a transparent proxy. It intercepts `call_tool()` and
  `list_tools()`, records timing and status, and sends spans asynchronously.
  If LangSight is unreachable, your agent works normally — nothing breaks.
</Note>

### Using a framework? Even simpler.

<Tabs>
  <Tab title="LangChain / LangGraph">
    ```python
    from langsight.integrations.langchain import LangSightLangChainCallback

    langsight = LangSightClient(url="http://localhost:8000")
    callback = LangSightLangChainCallback(client=langsight, agent_name="my-agent")

    # Pass as a callback — all tool calls traced automatically
    agent.invoke({"input": "..."}, config={"callbacks": [callback]})
    ```
  </Tab>
  <Tab title="CrewAI">
    ```python
    from langsight.integrations.crewai import LangSightCrewAICallback

    langsight = LangSightClient(url="http://localhost:8000")
    callback = LangSightCrewAICallback(client=langsight)

    crew = Crew(agents=[...], tasks=[...], callbacks=[callback])
    crew.kickoff()
    ```
  </Tab>
  <Tab title="Pydantic AI">
    ```python
    from langsight.integrations.pydantic_ai import langsight_tool

    langsight = LangSightClient(url="http://localhost:8000")

    @langsight_tool(client=langsight, server_name="postgres-mcp")

        return await session.call_tool("query", {"sql": sql})
    ```
  </Tab>

## 3 (optional). Add prevention guardrails

Prevent runaway loops and budget overruns with a few extra parameters:

```python
client = LangSightClient(
    url="http://localhost:8000",
    loop_detection=True,   # stop if agent loops the same call 3x
    max_steps=25,          # hard stop at 25 tool calls per session
    circuit_breaker=True,  # auto-disable servers after 5 consecutive failures
)
```

See [Prevention Guardrails](/sdk/python#prevention-guardrails) for full configuration options.

### Multi-agent tracing

If your agent hands off to sub-agents, LangSight captures the full tree:

```python
from langsight.sdk import LangSightClient

langsight = LangSightClient(url="http://localhost:8000")
trace_id = "trace-abc"
session_id = "sess-002"

# Orchestrator wraps its MCP client
orchestrator = langsight.wrap(
    mcp, server_name="jira-mcp",
    agent_name="orchestrator", trace_id=trace_id, session_id=session_id,
)

# create_handoff emits a handoff span with explicit target_agent_name
handoff = langsight.create_handoff(
    "orchestrator", "research-agent",
    trace_id=trace_id, session_id=session_id,
)

# wrap_child_agent pre-sets parent_span_id and agent_name from the handoff
researcher = langsight.wrap_child_agent(
    mcp, "confluence-mcp", "research-agent", handoff,
)
```

The dashboard renders this as a single connected tree across all agents.
